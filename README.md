## HOW TO USE
### USAGE:

    arch-pkgbuild-parser [FLAGS] --pkgbuild <PKGBUILD_FILE>
###
### FLAGS:
     -m, --makedepends    makedepends

     -h, --help           Prints help information

     -V, --version        Prints version information


### OPTIONS:
     -p, --pkgbuild <PKGBUILD_FILE>    PKGBUILD

## HOW TO BUILD
    cargo build
